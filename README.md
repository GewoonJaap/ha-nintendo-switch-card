## Installation

### Prerequisites

You need at least one [steam integration]() to use with this card.

```yaml
frontend:
  ... # your configuration.
  themes: !include_dir_merge_named themes
  ... # your configuration.
```

### HACS

1. Go to the Community Store.
2. Search for `steam card`.
3. Press `Install`.
